Orthogonal set

Definition:

Let SSS be a subset of Rn\mathbb{R}^nRn.

(a) SSS is said to be an orthogonal set if any two vectors in SSS are orthogonal.

(b) Furthermore, if every vector in SSS has unit length (i.e. norm 1), then SSS is said to be an orthonormal set.

Clearly, we can get an orthonormal set by normalizing each vector of an orthogonal set.
